# Form

Form submission with get and post requests.


# Pre-requisites
- Install [Node.js](https://nodejs.org/en/)


# Getting started
- Clone the repository
```
git clone https://github.com/CJThornburg/Form.git
```
- Install dependencies
```
cd FEForm
npm install
```



- Run the project
```
npm start
```
  Navigate to `http://localhost:2000`
